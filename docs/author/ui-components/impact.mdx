## Overview

You can use one or multiple impact components to present the numeral value of a specific statistic through the large typography and card-like style.

### Impact

Impact is used to display key information.

<Impacts>
  <Impact
    image={'/img/author/placeholder-icon-impact-40x40.png'}
    title="Number"
    label="Metric name"
  />
</Impacts>

### Impacts

A group for impact components.

<Impacts>
  <Impact
    image={'/img/author/placeholder-icon-impact-40x40.png'}
