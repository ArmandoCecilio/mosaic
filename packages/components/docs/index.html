Digital Platform Developer Components

This is the home of Digital Platform components.

The components conform to the Digital Platform Developer Design language, as specified by Developer Design Spec

The Developer Design language is a local design system built upon a JPM UI Toolkit foundation.

Features
  1. Components are theme-able and by default support light/dark modes.
  2. Components support both the Markdown use-case and stand-alone usage.

Labs

The src/labs directory contains 'ALPHA' components which are NOT yet ready for use in production.
It's an experimentation area, some of these components might never be used or are in-complete.

Do not use any 'lab' components in a Production environment

Contact the Developer Dev Team if you wish to learn more.

Contribution / Feature Requests

As a team with limited resources, we direct all feature requests to our Developer Product Team

We consider Developer an open platform and pull-requests are welcome.

Refer to our Contribution Guide
