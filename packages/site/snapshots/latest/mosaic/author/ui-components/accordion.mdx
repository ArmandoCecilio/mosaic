## Overview

As content authors, you can use an accordion component to invite users to:

- **Discover more information about a product**. Usually, there are a series of panes that contain
  summary content with an image to give the product appeal. Users can click to show or hide content.
- **Show a list of FAQs**. In this way, users locate the relevant question first, and then expand to
  check the answer.
